# Версия - 2025.4 (Relise)
#### от 24.12.2025 (build 24122025)

## Исправления

- Поиск клиентов (контактов, сделок), Универсальный поиск - улучшены
- Корпоративный ууниверситет 
  - исправление отображения дерева каталогов, выбор загруженного файла для материала
  - исправление удаления Лекции, материала и пр.
- Карточка клиента: Учет (и вывод) всех сделок компании и дочерних организаций
- Отчет по оплатам: восстановлен фильтр по правам сотрудника
- Экспорт клиентов - исправлена должность у осн.контакта
- Почтовик: поиск отправителя в базе по почтовому адресу с приоритетом наиболее новой записи (прикрепление письма к наиболее свежему Контакту с одинаковыми email)
- Исправлен скрипт обновления

>
> <b class="red">Важно</b>
> - Рекомендуется выполнять обновление через терминал, если размер базы данных значителен, т.к. вносятся изменения в её структуру
> ```php
> php /path_to_crm/_install/update.php
> ```
>

## При наличии проблем

>
> <b class="red">Рекомендуем</b>
> - При проблемах с добавлением записей Клиентов, Контактов, Сделок рекомендуется выполнить скрипт
>   - из консоли:
> ```php
> php /path_to_crm/_install/fix_default_values.php
> ```
>    - или из браузера:
> ```html
> https://адрес_црм/_install/fix_default_values.php
> ```
